# Generated mockups (gitignored)

This folder is for storing AI-generated mockup images. These files are gitignored to keep the repository size manageable.

## Naming Convention

- `trailglass-mockup-[journey]-[version].png`
- Example: `trailglass-mockup-timeline-v1.png`
- Example: `trailglass-mockup-onboarding-v2.png`

## Organization

Group by journey type:
- Core journey (timeline → map → details → stats)
- Onboarding flow
- Photo features
- Settings/configuration
- Other feature-specific flows
